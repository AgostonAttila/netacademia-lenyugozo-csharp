# Netacademia Lenyûgözõ C# tanfolyam

https://app.netacademia.hu/Tanfolyam/2018csharp-lenyugozo-c-programozas

Ezen a tanfolyamon mindenféle érdekes dologgal foglalkozunk majd, demonstrálandó, hogy aki programozni akar, annak miért érdemes a C#-pal kezdenie. Elsõ lépésként egy csevegõ robotot hozunk létre és állítunk üzembe teljesen nulláról olymódon, hogy közben a teljesen kezdõk számára is elmagyarázunk minden lépést! Ez inkább egy mutatom, hogy mit tudok tanfolyam, a kíváncsiak elég, ha nézik, az elszántak azonban a nagy részét meg is fogják tudni valósítani.

Készítsünk csetbotot!
